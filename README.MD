## Типа данных

* **Auto** — Автоматическое значение, интерпретируется конечной логикой
* **Undefined** — Обозначает, что элемент не определён
* **Nil** — Пустое значение
* **None** — Значение отсутствует
* **Boolean** — Логический, **true** | **false**
* **String** — Строка, **"" '' ``**
* **Object** — Объект
	* **Array** — Массив, **[item, item]**
	* **Dictionary** — Словарь, **{key: value}**
	* **Table** — Таблица
	* **Callable** — Вызываемый объект
		* **Function** — Функция
		* **Procedure** — Процедура
* **Quantity**
	* **Mumber** — Число
		* **Integer** — Целое число, **1**
		* **Float** — Дробное число, **1.0**
	* **Unit** — Единица измерения
		* **Length** — Длина, **1m** | **1cm** | **1ft** …
